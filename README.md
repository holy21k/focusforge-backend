# 🔨 FocusForge

**Forge Your Discipline. Build Your Legacy.**

An intelligent habit and task management system powered by AI predictions and real-time behavior analysis.

---

## Features

- 🤖 **AI Habit Coach** - Personalized coaching based on your behavior data
- 📊 **Smart Predictions** - Success probability for each habit with optimal timing
- 🔥 **Streak System** - Track consecutive completions and build consistency
- 💬 **Conversational AI** - Chat with your AI coach about habits and progress
- ⚔️ **Task Management** - Organize tasks with priority and completion tracking
- 🎨 **Personalization** - Theme, language, notifications, and goal settings

---

## Tech Stack

- **FastAPI** - High-performance Python web framework
- **MongoDB** - Flexible document database
- **Docker** - Containerized deployment
- **JWT** - Secure authentication

---

## Quick Start

```bash
git clone https://github.com/yourusername/focusforge-backend.git
cd focusforge-backend
docker-compose up --build
```

Access API docs at: `http://localhost:8000/docs`

---

## Core Principles

1. **Consistency Over Intensity** - Small daily actions build lasting change
2. **Data Drives Decisions** - Every completion improves AI predictions
3. **AI Assists, You Decide** - Suggestions are just suggestions
